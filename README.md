# Calculadora_asm
Projeto de uma calculadora desenvolvida em linguagem de máquina assemble

Este projeto deve conter as quatro operações primordiais: SOMA, SUBTRAÇÃO, MULTIPLICAÇÃO e DIVISÃO.

## Características do programa
- As operações de multiplicação e divisão não poderão utilizar as instruções MUL e DIV. Utilizar algoritmos que trabalham com soma e deslocamentos, para a multiplicação, e subtração e deslocamento para a divisão.  
- O resultado da operação poderá ser de um número de até 2 dígitos e poderá ser também negativo. Prever a impressão destes casos.  
- O programa deverá solicitar os 2 números e a operação, e devolver o resultado.  
- A interface com o usuário deverá ser a mais confortável possível. 
- Comentários são obrigatórios
